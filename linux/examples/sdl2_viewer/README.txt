JPEGDEC SDL2 Viewer
-------------------
This example program shows how to view JPEG images with JPEGDEC and
display them in a SDL2 window. The window waits for the user to press ESC
or click on the close button.

Requirements:
-------------
SDL2
Use the following command to install it on Linux:
 - sudo apt install libsdl2-dev

On MacOS:
brew install sdl2
